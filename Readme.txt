Algoritmo de Tomasulo - Autores : Arthur Estevão e Marcelo Ferreira

A presente pasta contém todo o projeto da prática 3 da disciplina de Laboratório de Arquitetura e Organização 
de Computadores II. Além disso, apresenta-se o relatório sobre o desenvolvimento do projeto, juntamente com a pasta
imagens que contém todas as imagens utilizadas no relatório.

Para realizar a simulação da mesma forma que fizemos no relatório:
	-Abra o projeto no Quartus II (el-tomasulo/bin/tomasulin.qpf)
	-Compile o projeto
	-Abra a simulação no ModelSim através do Quartus II
	-Na aba Simulate, selecione Start Simulation
	-Selecione Design - tomasulim
	-Selecione Libraries, em Search Libraries First (-Lf) -> Add -> altera_mf_ver ->Ok
	-Clique em Ok
	-Para carregar a simulação com a mesma organização que a apresentada, vá em File -> Load -> Macro File
	-Busque pelo diretório do projeto (el-tomasulo), entre na pasta data -> macro-files -> selecione Tomasulim-EX-03.do
	-Dê Run All
